Infinite Rockets (Alt)
POKE 6FCD,21
This alters the line:
6FCD CALL 6BE7
To:
6FCD LD HL,6BE7
So 6BE7 is never called (and HL is overwritten in the following command).

To:
6E34 LD A,(6BE7)
So 6BE7 is never called (and A is overwritten in the following command).
